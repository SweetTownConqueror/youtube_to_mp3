yt2mp3.py is a script I get at the following address:
https://gist.github.com/benzap/90ff22790bc0a9c6fd2902e91da4baef.

It allows to download from one or many youtube url the audio mp3 of the video.

My script myYtToMp3 aows to take a file in parameter, that contains a list of title to download
(it get the first youtub url linked to the title and calls the previous script in the aim to download it).

The file has to be named myList.txt (or you can modify yourself the name in the script) and each line has to
contain one title.

example:

rihanna man down
lp lost on you
...

Titles are going to be downloadded in the folder where the script is.
