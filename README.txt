yt2mp3.py dont j'ai récupéré le code à l'adresse suivante:
https://gist.github.com/benzap/90ff22790bc0a9c6fd2902e91da4baef

permet de télécharger une url youtube en format audio mp3.

Mon script myYtToMp3.py permet de prendre en parametre un fichier qui contient une liste de titres à
télécharger ( il va récupérer la premiere url de youtube lié à ce titre et appeler le script précédent
pour la télécharger)

Le fichier doit s'appeler mylist.txt, et etre de la forme 1 ligne = 1 titre

exemple:

rihanna man down
lp lost on you
...

les titres seront téléchargés dans le dossier ou se situe le script
